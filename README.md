# -neelimakothapalli11-droid-EDGE-TO-CLOUD-HEAT-MONITORING-FOR-INDUSTRIAL-ENVIRONMENTS-
**Edge-to-Cloud Heat Monitoring for Industrial Environments** uses **LM35**, **MQ2**, and **ESP-01** to monitor temperature and gas levels in real time. Data is processed at the edge and uploaded to **ThingSpeak** for remote monitoring, visualization, and alerts, helping prevent overheating, gas leaks, and industrial hazards.
